## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/BigAis/project-allecc.git
cd project-allecc
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Deployment

This project is configured for deployment to GitHub Pages.

1. Update the `base` property in `vite.config.js` with your repository name.

2. Build and deploy:
```bash
npm run deploy
```

3. Configure GitHub Pages in your repository settings to use the `gh-pages` branch.

4. Your site will be available at `https://bigais.github.io/project-allecc/`
